# 予定ちゃん - 製品要件定義書

## 製品ビジョン
スマホで簡単に予定調整ができるウェブアプリ。ログイン不要で、名前と行ける日程をタップするだけで回答できる。

## ターゲットユーザー
- 飲み会や遊びの予定を調整したい人
- ログインが面倒な人
- スマホで手軽に使いたい人

## 主要機能

### 1. イベント作成機能
作成者はログイン不要で以下を設定できる：

| 項目 | 必須 | 説明 |
|------|------|------|
| タイトル | ○ | イベント名 |
| 候補日時 | ○ | 日付 + 時間帯（複数選択可） |
| 場所 | - | 開催場所 |
| 説明 | - | 補足情報 |
| 合言葉 | ○ | 編集時に必要なパスワード |

- カレンダーUIから日付をタップで選択
- 時間帯は「終日」または「開始時間-終了時間」を指定
- 作成完了後、共有用URLを発行

### 2. 回答機能
参加者はログイン不要で回答できる：

- 名前を入力
- 各候補日に対して **○ / △ / ×** で回答
  - ○: 参加可能
  - △: たぶん参加可能
  - ×: 参加不可
- 回答後も編集可能（同じ名前で上書き）

### 3. 結果表示機能
- 全員の回答を一覧表示
- 各候補日の ○/△/× の集計を表示
- 最適な日程をハイライト（○が多い日）

### 4. 管理機能
作成者向け（合言葉で認証）：

- 回答ページで「編集する」ボタンをタップ
- 合言葉を入力して認証
- 認証後、以下が可能：
  - イベント情報の編集
  - 候補日の追加・削除
  - イベントの削除

## 非機能要件

### UI/UX
- **デザイン**: ミニマル、白基調でシンプル
- **レスポンシブ**: スマホファースト
- **操作性**: タップのみで完結、スワイプ不要

### パフォーマンス
- 初回表示: 2秒以内
- 操作レスポンス: 0.5秒以内

### セキュリティ
- XSS/SQLインジェクション対策

## 画面一覧

| 画面 | パス | 説明 |
|------|------|------|
| トップ | `/` | アプリ紹介 + 新規作成ボタン |
| 作成 | `/new` | イベント作成フォーム |
| 回答・管理 | `/e/[id]` | 参加者が回答 + 合言葉で編集モード |

## URL構成

```
/                       - トップページ
/new                    - イベント作成
/e/[eventId]            - イベント詳細・回答・管理ページ
```

- `eventId`: 公開ID（短いランダム文字列、例: `abc123`）
- 管理機能は同じページ内で合言葉認証後に表示

## 成功指標
- イベント作成から共有URLコピーまで1分以内で完了できる
- 回答は30秒以内で完了できる
