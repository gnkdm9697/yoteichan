# Requirements: 回答者名のローカルストレージ保存

## 概要
回答者が一度入力した名前をブラウザのローカルストレージに保存し、次回以降は自動入力する機能を追加する。

## ユーザーストーリー
- 回答者として、一度名前を入力したら次回以降は入力を省略したい
- 複数のイベントに回答する際、毎回名前を入力するのは面倒

## 要件
1. フォーム送信成功時に名前をローカルストレージに保存する
2. 次回アクセス時に保存された名前を自動入力する
3. ユーザーは自動入力された名前を編集可能

## 制約
- 既存の最大50文字制限は維持
- クリア機能は不要（シンプルに）
- SSR対応が必要（Next.js）

## 受け入れ基準
- [ ] 名前を入力してフォーム送信後、ローカルストレージに保存される
- [ ] ページリロード後、保存された名前が自動入力される
- [ ] 別のイベントページでも同じ名前が自動入力される
- [ ] 編集モード（initialName指定時）では復元しない
