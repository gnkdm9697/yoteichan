# Phase 3: 編集・仕上げ - 要件

## ゴール
合言葉認証による編集機能を実装し、デプロイして公開する。

## 前提
- Phase 1, 2 が完了していること

## 完了条件
- [ ] 合言葉を入力して編集モードに入れる
- [ ] イベント情報を編集できる
- [ ] 候補日を追加・削除できる
- [ ] イベントを削除できる
- [ ] Vercel にデプロイされている
- [ ] 本番環境で動作確認済み

## スコープ

### 含むもの
- 合言葉認証機能
- 編集モードUI
- イベント更新API
- イベント削除API
- UI/UX の仕上げ
- Vercel デプロイ

### 含まないもの
- ユーザー登録/ログイン機能
- 通知機能
- 多言語対応

## 機能詳細

### 1. 合言葉認証
- 「編集する」ボタンタップ → モーダル表示
- 合言葉入力 → 検証API呼び出し
- 正しい → 編集モードに切り替え
- 間違い → エラーメッセージ表示

### 2. 編集モード
- ヘッダーに「編集モード」表示
- イベント名の編集
- 場所・説明の編集
- 候補日の追加/削除
- 「変更を保存」ボタン
- 「編集を終了」ボタン
- 「イベントを削除」ボタン

### 3. 候補日の編集
- 既存候補日の削除（×ボタン）
- 新規候補日の追加
- ※既に回答がある候補日を削除する場合の警告

### 4. イベント削除
- 確認ダイアログ表示
- 削除後はトップページへリダイレクト

### 5. UI/UX 仕上げ
- ローディング表示
- エラーハンドリング
- トースト通知
- アニメーション
- フォント・余白の調整

### 6. デプロイ
- Vercel プロジェクト作成
- 環境変数設定
- 本番環境での動作確認
