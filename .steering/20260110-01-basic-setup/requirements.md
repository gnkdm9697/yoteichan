# Phase 1: 基本機能 - 要件

## ゴール
プロジェクトの初期設定とイベント作成機能を実装する。

## 完了条件
- [ ] Next.js プロジェクトが動作する
- [ ] Supabase に接続できる
- [ ] イベントを作成できる
- [ ] 作成したイベントのURLが発行される

## スコープ

### 含むもの
- Next.js プロジェクト初期化
- Tailwind CSS 設定
- Supabase プロジェクト作成・テーブル作成
- トップページ（`/`）
- イベント作成ページ（`/new`）
- 作成完了モーダル
- イベント詳細ページの骨組み（`/e/[id]`）

### 含まないもの
- 回答機能
- 編集機能
- 結果集計表示

## 機能詳細

### 1. トップページ
- アプリ名「予定ちゃん」表示
- 「予定を作成する」ボタン → `/new` へ遷移

### 2. イベント作成ページ
- タイトル入力（必須）
- 合言葉入力（必須）
- カレンダーから候補日選択（複数可）
- 各候補日の時間帯設定（終日 or 時間指定）
- 場所入力（任意）
- 説明入力（任意）
- 「イベントを作成」ボタン

### 3. 作成完了モーダル
- 共有用URL表示
- URLコピーボタン
- 「イベントを見る」ボタン

### 4. イベント詳細ページ（骨組み）
- イベント情報表示のみ
- 回答機能はPhase 2で実装
