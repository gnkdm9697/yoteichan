# Phase 2: 回答・表示機能 - 要件

## ゴール
参加者が回答でき、全員の回答を一覧表示できるようにする。

## 前提
- Phase 1 が完了していること

## 完了条件
- [ ] 名前を入力して回答できる
- [ ] ○/△/× で各候補日に回答できる
- [ ] 全員の回答が一覧表示される
- [ ] 各候補日の集計が表示される
- [ ] 同じ名前で回答を上書きできる

## スコープ

### 含むもの
- 回答フォーム（名前 + ○△× 選択）
- 回答一覧テーブル
- 集計表示（○/△/× の人数）
- 回答の保存・更新API

### 含まないもの
- イベント編集機能（Phase 3）
- 合言葉認証（Phase 3）

## 機能詳細

### 1. 回答フォーム
- 名前入力フィールド
- 各候補日に対して ○/△/× ボタン
  - ○: 参加可能（緑）
  - △: たぶん参加可能（オレンジ）
  - ×: 参加不可（赤）
- 「回答する」ボタン
- 回答成功時のフィードバック

### 2. 回答一覧テーブル
- 横軸: 候補日
- 縦軸: 参加者名
- セル: ○/△/× を色付きで表示
- 最下部に集計行

### 3. 集計表示
- 各候補日ごとに ○/△/× の人数を表示
- ○ が最も多い日をハイライト（オプション）

### 4. 回答の上書き
- 同じ名前で再度回答すると上書き
- 「〇〇さんの回答を更新しました」メッセージ
